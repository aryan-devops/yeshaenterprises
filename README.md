# Yesha Enterprises — Biofloc Fish Farming Solutions

Yesha Enterprises is India's premier distribution hub for high-grade Biofloc fish farming infrastructure. We specialize in premium HDPE pond liners, ring blowers, portable fish tanks, and generators. Our platform offers a seamless product catalog with detailed specifications and WhatsApp inquiry integration for a premium experience.

## 🌟 Key Features

- **Premium Product Details**: Dedicated pages for every product with technical specifications and smooth hover-zoom galleries.
- **Dynamic Catalog**: Real-time product management powered by Supabase.
- **Smart Inquiry**: Integrated "Send Inquiry on WhatsApp" functionality with automated messaging.
- **Theme Support**: Seamless switching between Light and Dark modes with theme-aware branding.
- **Premium UX**: High-performance preloader, smooth animations, and ultra-rounded modern design.
- **Admin Panel**: Comprehensive dashboard for managing products, images, and technical data.

## 🛠️ Technology Stack

- **Frontend**: React 19, Vite, Vanilla CSS (Premium UI System)
- **Backend/Database**: Supabase (PostgreSQL & Storage)
- **Icons**: Lucide React
- **Routing**: React Router 7

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/aryan-devops/chatting-app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd yeshaenterprises
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Create a `.env.local` file in the root and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Reusable UI components (Navbar, Footer, Preloader, etc.)
- `src/pages`: Main application pages (HomePage, ProductDetailPage, etc.)
- `src/lib`: External service configurations (Supabase client)
- `src/assets`: Static assets including the official SVG logo.
